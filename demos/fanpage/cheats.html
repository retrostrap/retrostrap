How to enter a code

At the hangar screen (unless a code says otherwise), press Esc to pause, type the code in CAPITALS, then press Enter. A good code chirps. A bad code does nothing and judges you.

C:\SQUEST>
hangar> _pause_
hangar> BIGCHEESE
>> all weapon systems ONLINE. have fun, cadet.

hangar> _resume_
1 cheat active, 0 regrets
