THANK YOU for evaluating To-Do List 98, the friendly task manager that keeps
your list on YOUR OWN COMPUTER. No server. No upload. No modem required after
the page has loaded. Close the browser, come back tomorrow, your list is
waiting.

FILES
  index.html .... the program
  help.html ..... the help file (Contents / Shortcuts / Register)
  app.js ........ the whole program, in plain JavaScript, no build required

HOW YOUR LIST IS SAVED
  In this browser, on this machine, under the key "rs-demo-todo". Move to a
  different computer and you begin again with a fresh, hopeful, empty list.

KEYBOARD
  Enter .... add the task you typed
  Space .... toggle the selected task done / undone
  Del ...... delete the selected task (it asks first)
  Up/Down .. move the selection
  Esc ...... close the dialog box on screen
