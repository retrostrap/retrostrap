What this program does

To-Do List 98 keeps a list of things you mean to do. It saves that list to this very computer, no server, no upload, no modem required after the page has loaded. Close the browser, come back tomorrow, your list is waiting.

Using it

  • Type a task in the box and press Enter (or click Add).
  • Tick the box next to a task to mark it done. The progress bar fills as you go.
  • Click a task to select it, then use Delete or Mark done from the toolbar or the Edit menu.
  • Clear completed sweeps away every finished task at once, it asks first.

Your list lives in this browser on this machine. Use a different computer and you start with a fresh, hopeful, empty list.

Keyboard shortcuts

Select a task first (click it), then the keys below work whenever the typing box is not focused.

Keyboard shortcuts
KeyWhat it does
EnterAdd the task you just typed
SpaceToggle the selected task done / undone
DelDelete the selected task (asks first)
↑ ↓Move the selection up and down
EscClose the dialog box on screen

Rumour has it F1 opens this very page. Rumour is generous; the button just feels correct.
