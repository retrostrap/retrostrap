HOW IT IS BUILT
  plain HTML dressed up with retrostrap (the files in ../../dist/). the pink
  is the "kawaii" theme. the sparkles and the little hearts that follow your
  mouse are retrostrap widgets, and so are the smilies -- when you type :) it
  turns into a real pixel face, but if you copy-paste it you still get :) so
  it is honest. nothing on this page phones home or tracks you. gross.

  the about page turns on "comfy mode" (one class on the html tag) so the
  writing is a little bigger and cozier. try it!!
